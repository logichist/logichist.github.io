# logichist official home

(주)헛소리 그룹 산하 (주)로지키스트 공식 홈페이지입니다.

## 소개

1998년 여름 천보산 뒤편 약수터에서 발기한 (주)헛소리는 회장님의 친목회 성격으로 운영되어 왔습니다.

2019년 (주)로지키스트 법인을 설립 후 막후실세로 잡일을 다 하는 중입니다.

사실 회장인 [haherhohu](https://github.com/haherhohu)는 자기가 필요한거만 만들고 있습니다.

혹 가끔 필요하다고 하는 게 있다면 만들 수도 있습니다.
